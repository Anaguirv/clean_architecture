# Clean Architecture

## Entorno virtual

### Instalar entorno virtual

```bash
pip install virtualenv
```
### Crear entorno virtual
```
python<version> -m venv <virtual-environment-name>
```
 ### Ejemplo
```
 mkdir projectA
 cd projectA
 python3 -m venv venv
 ```
### Activar entorno
```
source venv/bin/activate
```
### Revisar paquetes instalados en ambiente virtual
```
pip list
```
